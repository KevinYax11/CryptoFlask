Cómo Funciona

1

Genera tus Claves

Crea un par de claves RSA (pública y privada) con un nombre identificador único. Las claves se almacenan de forma segura en formato PEM.

2

Cifra o Firma

Usa la clave pública para cifrar mensajes destinados al propietario de la clave privada, o usa tu clave privada para firmar mensajes que quieras autenticar.

3

Descifra o Verifica

Descifra mensajes con tu clave privada para leer contenido confidencial, o verifica firmas digitales con la clave pública del firmante para validar autenticidad.

Características Técnicas

Algoritmo RSA-2048

Utiliza claves de 2048 bits que proporcionan un alto nivel de seguridad para datos sensibles.

Padding OAEP

Optimal Asymmetric Encryption Padding con SHA-256 para cifrado seguro y resistente a ataques.

Firma RSA-PSS

Probabilistic Signature Scheme con SHA-256 para firmas digitales seguras y verificables.

Formato Base64

Todos los mensajes cifrados y firmas se codifican en Base64 para fácil transferencia y almacenamiento.
